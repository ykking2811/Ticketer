# Ticketer

## Running Locally

To run this application locally, you just need [Node.js](https://nodejs.org/) installed on your computer.

### Quick Start (Recommended)

**On Windows:**
Simply double-click the `start.bat` file in the project folder. It will automatically check for Node.js, install any required dependencies, and start the server.

**On Mac/Linux:**
Open a terminal in the project folder and run the `start.sh` script:
```bash
chmod +x start.sh
./start.sh
```

### Manual Start

If you prefer to run it manually using the terminal:

1. Open a terminal or command prompt in this directory.
2. Install the dependencies (only needed the first time):
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

The application will launch and tell you which local address to type into your web browser.
